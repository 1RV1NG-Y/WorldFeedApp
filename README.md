# WorldFeedApp
Mobile app version of [WorldFeed](https://github.com/1RV1NG-Y/WorldFeed) project made with react native

## installation

In the terminal, you can run:

### `npm install`


In the project directory, you can run:

### `npx expo start`

### `npm install react-native-paper`                                                                                                                       

### `npm install axios`
